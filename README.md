# 서울숲 보도자료 자동생성기

행사 정보를 입력하면 서울시 공식 보도자료 형식(□/○ 기호, 격식체 문장)에 맞춰
완성된 보도자료를 자동으로 생성해주는 웹 앱입니다.

## 사용 방법

1. 행사명 / 일정 / 주요 프로그램 / 안내 / 핵심메시지(필수 항목) 입력
2. 부제 / 배경 / 참여 / 인용문(선택 항목)은 필요할 때만 입력
3. "샘플로 채우기" 버튼으로 예시 데이터를 채워 바로 확인 가능
4. "보도자료 작성" 버튼 클릭 → 완성된 보도자료가 화면에 표시
5. "복사하기" 버튼으로 결과 텍스트 복사

## 파일 구성

- `index.html` — 페이지 구조
- `styles.css` — 스타일
- `app.js` — 입력 검증 및 보도자료 생성 로직
- `vercel.json` — Vercel 배포 설정
